# Node.js Fullstack Application

This project is a fullstack web application built using **Node.js**, **Express.js**, **EJS**, and **MySQL**. The application includes essential features such as user login/logout, profile management, job posting, and job request handling.

## Features

- **User Authentication:**
  - **Login**: Users can log in with their credentials.
  - **Logout**: Users can log out to terminate their session.
  - **Profile Update**: Users can update their profile details.

- **Job Management:**
  - **Create Job Post**: Admins or users with appropriate roles can create job posts.
  - **Delete Job Post**: Admins or authorized users can delete a job post.
  - **Apply for Job**: Users can apply for job postings.
  - **Accept Job Request**: Admins or employers can accept job applications.
  - **Reject Job Request**: Admins or employers can reject job applications.

## Tech Stack

- **Node.js**: JavaScript runtime environment.
- **Express.js**: Web framework for Node.js.
- **EJS**: Templating engine for rendering HTML views.
- **MySQL**: Relational database management system for storing data.
